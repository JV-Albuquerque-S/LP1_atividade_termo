## 🛠️ Como Compilar e Executar

Cada projeto está isolado em sua própria pasta. Para compilar e rodar via terminal, siga o modelo abaixo:

### Passo a Passo:

1. **Navegue até a pasta do exercício:**
   ```bash
   cd [pasta]
   ```

2. **Compile o código-fonte:**
   ```bash
   gcc [arquivo].c -o programa
   ```

2. **Execute o arquivo gerado:**
  - Windows
     ```bash
     ./programa.exe
     ```
  - Linux/Mac
     ```bash
     ./programa
     ```
